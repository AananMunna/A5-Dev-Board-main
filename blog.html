Question-1: What are the different ways to select an element in the DOM?

একটি নির্দিষ্ট ID চাইলে → getElementById() একই ক্লাসের একাধিক এলিমেন্ট চাইলে → getElementsByClassName() একই ট্যাগের একাধিক এলিমেন্ট চাইলে → getElementsByTagName() CSS সিলেক্টরের মতো ব্যবহার করতে চাইলে → querySelector() বা querySelectorAll()

Question-2: What is the difference between innerHTML, InnerText, and textContent?

innerHTML → HTML সহ নেয় innerText → শুধু দেখা যায় এমন টেক্সট নেয় textContent → সব টেক্সট নেয়, এমনকি লুকানো থাকলেও

Question-3: What is event delegation in the DOM?

Event Delegation হলো একটি ইভেন্ট হ্যান্ডলারকে প্যারেন্ট এলিমেন্টের সাথে সংযুক্ত করা, যাতে তার চাইল্ড এলিমেন্টগুলোর ইভেন্টও পরিচালিত হতে পারে।

Question-4: What is event bubbling in the DOM?

Event Bubbling হলো একটি ইভেন্ট যখন চাইল্ড এলিমেন্টে ট্রিগার হয়, তখন তা অটোmatically তার প্যারেন্ট এবং উপরের এলিমেন্টগুলোতে ছড়িয়ে যায়।

Question-5: How do you create, add, and remove elements using JavaScript?

নতুন এলিমেন্ট তৈরি document.createElement("tag") এলিমেন্ট যোগ appendChild(), prepend(), insertBefore() এলিমেন্ট মুছতে remove(), removeChild()
